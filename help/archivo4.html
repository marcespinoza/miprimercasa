• Nuevo propietario

Agrega un nuevo propietario sin ninguna propiedad asignada

  • Eliminar propietario

Elimina el propietario seleccionado. Si tiene alguna propiedad asiganda tambien se elminaran los detalles de pago. Se realiza una baja fisica, es decir se elimina al propietario de la base de datos.

  • Editar propietario

Editar los datos del propietario y su referente

  • Dar de baja

Se realiza una baja logica, es decir, el propietario no se elimina de la base de datos y permanece en la lista de clientes en color rojo. Los detalles de pagos realizados siguen siendo accesibles para el usuario. Al realizar una baja, la propiedad se libera, siendo posible asignar a un nuevo cliente.

  • Detalle de pago

Permite ver detalles de las cuotas abonadas.

  • Cambiar propietario

Permite transferir el lote y todas las cuotas abonadas, a un nuevo cliente o a uno existente. El cliente anterior queda libre.

  • Asignar propiedad

Asigna un lote libre a un cliente.

  • Agregar propietario

Agrega un nuevo propietario a un lote con propietario ya asignado.

  • Cambiar propietario

Permite transferir el lote y todas las cuotas abonadas, a un nuevo cliente o a uno existente. El cliente anterior queda libre.
